# Pedro Henrique Troglio Hochsprung — Currículo Online

Repositório do meu currículo online publicado via **GitHub Pages**.

🌐 **Acesse em:** https://goverllly.github.io/pedro-hochsprung.github.io/

## Sobre o site

Página de currículo profissional desenvolvida com HTML5, CSS3 e JavaScript puro, sem dependências de framework. Inclui:

- Animação de partículas no background
- Efeito de digitação (typing effect) no título
- Timeline animada de experiências profissionais
- Barras de habilidades com animação ao scroll
- Design responsivo para mobile e desktop
- Esquema de cores dark mode profissional

## Estrutura do projeto

- `index.html`: estrutura semântica da página
- `assets/css/style.css`: estilos, layout e responsividade
- `assets/js/main.js`: interações, animações e comportamento da navegação

## GitHub Pages
